# 存在の証明 - 3DCG Works Archive 🌌

> **FRYX (Takumi Furuya)** の3DCG作品集サイト

メインサイト（[fryx404.github.io](https://fryx404.github.io/)）から3D作品を閲覧するためのサブサイト。

## 🎭 コンセプト

**存在論的アプローチ**による3DCG作品アーカイヴ。哲学的テーマを視覚化したダークミニマルデザインで、作品鑑賞に特化した体験を提供。

- **作品アーカイヴ**: 3DCG・NFT作品のギャラリー表示
- **イマーシブモーダル**: フルスクリーンの没入型作品詳細ビュー
- **ミニマル美学**: 余白と沈黙が織りなす美しさ
- **パーティクル演出**: Canvas APIによる浮遊粒子システム

## 🌟 主要機能

### 🎨 視覚的特徴
- **ダークテーマUI**: 存在論的デザインシステム
- **パーティクルシステム**: Canvas APIによる50個の浮遊粒子
- **呼吸エフェクト**: 背景の微細な明滅（4秒サイクル）
- **グラスモーフィズム**: モーダル情報パネルの半透明ブラーサーフェス

### 📱 ユーザー体験
- **シングルページ**: index.html に全作品ギャラリーを集約
- **イマーシブモーダル**: フルスクリーン作品詳細（前後ナビ・キーボード対応）
- **リアルタイム時計**: 現在時刻の表示
- **レスポンシブ対応**: 全デバイスでの最適表示（768px・480pxブレークポイント）
- **メインサイトへの戻りリンク**: 左上に常時表示

### ⚡ インタラクティブ機能
- **作品ナビゲーション**: ←→キーまたはボタンで前後の作品に移動
- **ESCキーで閉じる**: キーボードショートカット対応
- **スクロールトップ**: ワンクリック頂点回帰
- **動的統計表示**: 作品数の自動計算
- **外部リンク統合**: ArtStation・Behance直接リンク

## 🎯 作品データ管理

### 作品の追加・編集
`scripts/works.js` の `WORKS_DATA` 配列に新しい作品を追加：

```javascript
{
    id: "unique-id",                    // 一意のID
    title: "作品名",                    // 作品タイトル
    description: "作品の哲学的記述...", // 詩的な説明文（HTMLタグ対応）
    technologies: ["Blender", "Photoshop"], // 使用技術配列
    duration: "制作期間",               // 開発期間
    type: "3DCG",                      // 作品カテゴリ
    status: "完成",                    // 完成|開発中|構想中
    year: 2025,                        // 制作年
    image: "images/works/filename.png", // 画像パス
    url: "https://..."                 // 外部リンク（ArtStation/Behance）
}
```

## 🎨 デザインシステム

### カラーパレット
```css
:root {
    /* 基本色 */
    --void: #000000;              /* 虚無の黒 */
    --existence: #ffffff;         /* 存在の白 */
    --liminal: #0a0a0a;          /* 境界のグレー */
    
    /* 透明度バリエーション（存在論的命名）*/
    --whisper: rgba(255, 255, 255, 0.03);     /* 囁き */
    --breath: rgba(255, 255, 255, 0.1);       /* 呼吸 */
    --pulse: rgba(255, 255, 255, 0.2);        /* 鼓動 */
    --manifestation: rgba(255, 255, 255, 0.4); /* 顕現 */
    --essence: rgba(255, 255, 255, 0.6);      /* 本質 */
    --transcendence: rgba(255, 255, 255, 0.9); /* 超越 */
}
```

### タイポグラフィ
- **メインフォント**: Noto Sans JP（和文サンセリフ）
- **セリフ**: Noto Serif JP（和文セリフ、見出し用）
- **ウェイト定義**: 100（weightless）〜900（transcendence-weight）

### アニメーション設定
```css
:root {
    --instant: 0.1s;        /* 瞬間 */
    --moment: 0.3s;         /* 一瞬 */
    --breath-cycle: 4s;     /* 呼吸サイクル */
    --existence-cycle: 8s;  /* 存在サイクル */
    --eternity: 20s;        /* 永遠 */
}
```

## 🎬 技術実装詳細

### JavaScript アーキテクチャ

#### main.js - パーティクル & 時計
- **ExistenceState**クラス: 初期化管理
- **ParticleSystem**クラス: Canvas粒子システム（50個の粒子）
- **Particle**クラス: 個別粒子の動作
- リアルタイム時計更新

#### works.js - ギャラリー & モーダル
- **CreationGarden**クラス: 作品表示・管理
- **WORKS_DATA**配列: 45作品の完全データ
- 作品データの動的レンダリング（年度順ソート）
- **イマーシブモーダル**: フルスクリーン詳細ビュー
  - グラスモーフィズム情報パネル
  - 前後ナビゲーション（ボタン + キーボード←→）
  - ESCキーで閉じる
  - 作品カウンター表示
- 統計情報の自動計算

### CSS アーキテクチャ
- **存在論的命名**: 哲学的概念に基づくCSS変数
- **グラスモーフィズム**: backdrop-filter による半透明パネル
- **レスポンシブ**: 3段階（1024px・768px・480px）
- **シネマティックアニメーション**: cubic-bezier によるスムーズな開閉

## 🔧 カスタマイズ方法

### 1. 作品の追加
`scripts/works.js` の `WORKS_DATA` 配列の先頭にオブジェクトを追加。

### 2. 色彩テーマ変更
```css
/* styles/main.css の :root 変数を編集 */
:root {
    --void: #your-background-color;
    --existence: #your-text-color;
}
```

### 3. アニメーション調整
main.js の `maxParticles` や CSS の `--breath-cycle` 等を変更。
